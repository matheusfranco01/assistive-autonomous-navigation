# Assistive Autonomous Navigation

Assistive navigation platform developed using ROS, LiDAR and Kinect sensors for autonomous mobile robot navigation and gesture-based control.

## Features

- 3D Mapping
- SLAM using RTAB-Map
- Obstacle Avoidance
- Path Planning
- Gesture Recognition
- Autonomous Navigation

## Hardware

- Pioneer P3DX
- LiDAR Sensor
- Microsoft Kinect

## Software

- ROS Noetic
- RViz
- Gazebo
- RTAB-Map
